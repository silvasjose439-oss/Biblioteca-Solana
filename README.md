Tienda de Laptops en Solana

Proyecto básico de un Solana Program desarrollado con Rust y Anchor en Solana Playground.
El programa implementa operaciones tipo CRUD para gestionar una tienda de laptops en la blockchain, permitiendo crear la tienda, agregar laptops, ver el inventario y eliminar laptops.

Este proyecto fue desarrollado como parte de la preparación para la certificación de Solana, con el objetivo de practicar el desarrollo de programas on-chain, manejo de cuentas PDA y uso del framework Anchor.

Uso del proyecto

Puedes importar el proyecto directamente en Solana Playground desde tu repositorio:

https://beta.solpg.io/

Luego solo debes:

Conectar tu wallet en Devnet.

Compilar el programa.

Ejecutar las pruebas para interactuar con la tienda de laptops.

Este proyecto sirve como ejemplo sencillo para aprender a construir aplicaciones descentralizadas (dApps) en Solana. 🚀
